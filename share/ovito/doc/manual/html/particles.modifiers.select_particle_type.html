Select particle type

This modifier selects particles of a certain type.

Parameters

Property

Controls what input property, for instance the chemical type or the structural type, should be used to select particles. The drop-down box lists all available source properties that store type information.

Types

The list of types defined in the source property. Check one or multiple types to let the modifier select the corresponding particles.
