Histogram

This modifier computes a histogram of the values of a particle or bond property.

Additionally, the modifier allows to select all particles or bonds whose property values lie within a certain interval.

Parameters

Input

Selects whether input property values should be taken from particles or bonds.

Property

The particle or bond property the histogram should be computed for.

Number of histogram bins

Controls the resolution of the generated histogram.

Use only selected particles/bonds

Restricts the histogram calculation to the subset of particles or bonds that are currently selected.

Select value range

Lets the modifier select all particles or bonds whose property value lies in the specified range.
