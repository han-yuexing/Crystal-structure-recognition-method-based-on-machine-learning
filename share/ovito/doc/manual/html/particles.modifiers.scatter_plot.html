Scatter plot

This modifier computes and displays a scatter plot for two particle properties. This type of plot can be used to study possible correlations between two particle properties. A scatter plot consists of one data point per input particle. The first property determines the X-coordinates and the second property the Y-coordinates of the points.

Optionally, the modifier can be used to select particles within a specified X- or Y-range.

Parameters

X-axis property

The per-particle property that serves as data source for the x-coordinates of data points.

Y-axis property

The per-particle property that serves as data source for the y-coordinates of data points.
